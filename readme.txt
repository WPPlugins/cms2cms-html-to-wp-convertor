Automated HTML to WordPress content migration plugin

== Description ==

You have a lot of content on your HTML website but you’ve considered that WordPress CMS suits you better. You start thinking of moving HTML website content to WordPress. You’re mostly interested in a quick and profound migration.

= Here’s how this plugin helps you do it: =
- Free Demo Migration helps see the migration outcome beforehand;
- 15 min is the average migration time (Note: timing depends on how many content items you’re going to move from HTML to WordPress);
- Migrate the content you want. Choose how you want it to be and look on WordPress (Data ‘Mapping’ option);
- Complete data migration price depends on the content volume you’re about to migrate to WordPress.

= What you can move from HTML to WordPress: =
*(Note that CMS2CMS moves only content data from HTML to WordPress)*
* Pages along with the content included there ;
* Images included in the content body;

= Preserved SEO rankings after the migration =
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing HTML website to the new WordPress website.

= Compatible with: WordPress 4.1.x - 4.8.x. =
- Live help is available. Our support team is here 24/7.
- When a custom migration is needed, CMS2CMS offers data migration service packages to order.

== Installation ==

1. Download the plugin zip file.
1. Upload it to your WordPress website.
1. Go to Admin -> Plugins, find “CMS2CMS Automated [Your Source CMS platform] to WordPress Migration Plugin” click on it.
1. Register an account.
1. Provide your Source website URL address.
1. Click ‘Verify Connection’.
1. You’ll be redirected to CMS2CMS website in order to proceed with your migration.
1. Map the data you’re up to move from the Source website to your Target WordPress.
1. Select additional options.
1. Run Free Demo Migration to preview the results first.
1. Check how much the transfer will cost you and go directly to the Complete migration step.
